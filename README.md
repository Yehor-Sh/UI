# UI

Обновлённый макет интерфейса доступен в папке `combined_ui`. Он содержит:

- `index.html` — единый экран, объединяющий боковое меню, общий обзор и детали сделки.
- `styles.css` — стилизация интерфейса с адаптивной версткой.
- `script.js` — минимальная интерактивность для переключения панелей и деталей сделок.

Откройте `combined_ui/index.html` в браузере, чтобы просмотреть собранный интерфейс и продолжить модернизацию.
